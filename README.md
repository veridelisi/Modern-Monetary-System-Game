# 💰 Money Game — Who Creates Money?

🎮 An interactive game built to explore one of the most misunderstood questions in economics:

## 👉 Who actually creates money?

This project turns monetary economics into a simple interactive experience.  
Instead of long theoretical explanations, the game helps users learn by playing.

---

## 🚀 Play the Game

**Live app:**  
[https://money-game.streamlit.app](https://money-game.streamlit.app)

---

## 🧠 What the Game Explores

- 💳 Do banks lend existing money, or create new money?
- 🏦 What is the real role of central banks?
- 📊 How does the modern monetary system actually work?
- ⚖️ Why is money creation so often misunderstood?

## 🎯 Why This Project Exists

Many people still believe:

- ❌ Central banks create most of the money
- ❌ Banks simply lend out existing deposits
- ❌ Money creation is easy to understand from textbooks alone

This game was created to challenge those assumptions in a more engaging way. 🎮
